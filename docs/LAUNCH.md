# WalkAway — Launch Runbook ($4.99, direct paid)

## Why not the Mac App Store
MAS requires the App Sandbox. This app cannot pass review under it:
- **Screen lock uses a private API** — `SACLockScreenImmediate` via `dlopen` of
  `login.framework` (`Locker.swift`). Auto-rejected (Guideline 2.5.1); there is
  no public macOS screen-lock API.
- **Apple Watch RSSI needs a subprocess** — `system_profiler SPBluetoothDataType`
  (`SystemBluetoothSnapshot.swift`). Sandbox forbids launching external
  executables, so the core feature returns nothing.
Neither has a sandbox-legal workaround. We ship **Developer ID + notarized**,
sold direct.

## Licensing (built, store-agnostic)
- Ed25519-signed offline keys. App embeds only the public key
  (`LicenseStore.publicKeyBase64`) and verifies keys offline → works with any
  storefront or hand-issued keys.
- **14-day full trial** from first launch. After expiry the app runs but
  auto-lock is disabled until a valid key is entered (`LockController` gate).
- Signing keypair: generated by `script/license_tool.swift generate-keys`.
  Private key lives in `secrets/license_keys.txt` (gitignored) — **never commit it**.

### Issue a key by hand
```
swift script/license_tool.swift issue "<PRIVATE_KEY_B64>" "buyer@example.com"
```
Give the resulting `payload.signature` string to the buyer; they paste it into
the menu → "Enter license key".

- [ ] Screenshots / demo GIF
- [ ] Auto-update via **Sparkle** (see below)

## Sparkle auto-update (not yet wired)
Direct-distribution apps need their own updater. To add:
1. Add Sparkle SPM dependency to `Package.swift`.
2. Generate Sparkle EdDSA keys (`generate_keys`), embed the public key + a
   `SUFeedURL` in Info.plist (extend `package_release.sh`).
3. Host an `appcast.xml` (GitHub Pages / S3); append each release's DMG with a
   signed enclosure on every `package_release.sh` run.
Deferred deliberately — it needs a hosted appcast URL and its own signing keys.

## Release mechanics
`VERSION=x.y.z BUILD_NUMBER=n ./script/package_release.sh` →
signed, notarized, stapled `dist/release/WalkAway-x.y.z.{dmg,zip}`. Then tag +
`gh release create` (see git history for the pattern).
